## [v0.3.0] - 2026-03-29

### Highlights

- Multi-window support - open multiple windows with independent grid layouts sharing the same session pool
- Pop-out sessions - expand any session to its own dedicated window
- Renamed from Nexus to PlexusOne Desktop

### Added

- Multi-window support with independent grid layouts per window ([`ff8d164`](https://github.com/plexusone/plexusone-app/commit/ff8d164))
- Pop-out session feature to open sessions in dedicated 1×1 windows ([`e50668d`](https://github.com/plexusone/plexusone-app/commit/e50668d))
- AppState singleton for shared state across windows ([`b04e7ea`](https://github.com/plexusone/plexusone-app/commit/b04e7ea))
- WindowStateManager for multi-window persistence ([`8ccbe27`](https://github.com/plexusone/plexusone-app/commit/8ccbe27))
- Multi-window state models (WindowConfig, WindowFrame, MultiWindowState) ([`0232b80`](https://github.com/plexusone/plexusone-app/commit/0232b80))
